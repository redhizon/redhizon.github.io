Favorite Programming Language

When you enter the university as a computer science student, expect that you will learn a lot of programming languages. At the time being, I know 5 languages: Python, C, Javascript, Java, and R. Among the 5, my favorite is the R programming language for the reason that the syntax is relatively easier to understand and type.

I love R becuase it is very close to the English language when compared to other languages like C and Java. It also allows users to create collections of things like arrays and vectors in a jiffy! You don't even have to worry about memory allocation or get confused with 0-indexing system of arrays. I'm not saying that this is the best programming language, but R provides you an easier way to analyze data by implementing abstract ideas and formula into code without having to think much about the syntax. It gives you the focus that you need in order to perform numeric and symbolic computations. To learn more about R,  click here.

Note that even though R is my favorite programming language, I am not saying that this is always my first choice for a project. Remember that it is okay to have a favorite programming language, but know that each language has its own strenghts and weaknesses, so choose your language accordingly.
